Text Formatting

There are three main types of text formatting. Those are bold, italics, and underline.

Inline vs. Block Elements

A block element moves the content to a new line. The paragraph and header elements are block elements. An inline element does not display its content on a new line. it keeps the content on the same line as the previous content. Bold, italics, and underline are all inline elements.

Vertical Text Spacing

There are two elements that help with vertical spacing between block elements. A line break element adds an empty line an moves the content down to a new line. its similar to the ENTER key in a Google/Word doc. A Horizontal rule is a line break with a horizontal bar across the page. Both of these are block elements

Attributes

Attributes are properties which can change an HTML element. Attributes are written in the opening tag of ANY html element. The most common attributes are src, href, class, id, and style.
