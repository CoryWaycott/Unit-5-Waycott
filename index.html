Text Formatting

There are three main types of text formatting. Those are bold, italics, and underline.

Inline vs. Block Elements

A block element moves the content to a new line. The paragraph and header elements are block elements. An inline element does not display its content on a new line. it keeps the content on the same line as the previous content. Bold, italics, and underline are all inline elements.

Images

An image element is an inline element which inserts pictures into the document. An image element is self-closing, which means it does not have a closing tag. it has one major attribute, src, which is a filepath that points to the images' location. Those filepaths can either be absolute or relative.

Simple Image Example

Image Attributes

There are 4 attributes commonly used for images. width and height are used to resize the images. alt allows to use text description of images for accessibility and broken images. title allows a hover-over text to appear.

Width and Height

The width and height can be measured in many different ways, but the more common ones are pixels and percentages. Using pixels, you set the specific size based on pixels. using percentages, you set the dimensions based on a percentage of the entire page. In either case setting the width alone respects aspect ration, while setting both does not.

Alt Attribute

EVERY picture should at least have the alt attribute. The alt attribute helps with screen readers for visually impaired and in the cases that the image is broken.

Relative vs. Absolute

An absolute filepath includes all the navigation to get to the server, folder and file. The examples above are all absolute filepaths. An absolute file path is used when the image is stored on a different server or device than the HTML document.

A relative filepath only includes folders and files. The example below is a relative filepath. A relative filepath is used when the images are stored alongside the HTML document.

Links

The anchor element is an inline element and is a way to include hypertext links, otherwise known as just links, in your document. An anchor is a reference point for other documents or files. It commonly has 3 attributes: href, target, and download.
